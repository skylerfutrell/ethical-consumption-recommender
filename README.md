# Ethical Consumption Recommender

## Overview
The Ethical Consumption Recommender is an AI-powered tool that suggests products based on sustainability, environmental impact, and ethical sourcing. It helps users make responsible purchasing decisions through data-driven insights.

## Features
- Recommends products based on environmental and ethical scores  
- Uses AI and machine learning for scoring and ranking  
- Interactive dashboard for visualizing recommendations  
- Stores product and user data in SQL for scalable analysis  

## Tech Stack
- **Python**  
- **SQL (PostgreSQL / SQLite)**  
- **FastAPI** (backend API)  
- **Pandas / NumPy / Scikit-learn** (data processing & AI)  
- **React** (frontend dashboard)  
- **Chart.js / Tailwind** (visualization & styling)  

## Getting Started

### Prerequisites
- Python 3.10+  
- Node.js 18+  
- PostgreSQL (or SQLite for MVP)
